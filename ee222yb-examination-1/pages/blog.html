What is robots.txt and how have you configured it for your site?

Robots.txt is a file located in the root folder that helps bots, like search engines, knowing which pages not to index. It is set up, so all bots avoids indexing contact.html

What is humans.txt and how have you configured it for your site?

Humans.txt is a meta text file that should be placed in the root folder containing information about the creators of the site. What is Open Graph and how do you make use of it?

Open graph was developed by Facebook and allows for easy integration between a webpage and social media. It decides what part of the website (e.g. an article) should be shared on the social media platform, and links it to user data, so that e.g. facebook can track how many times an article has been shared.

To use it on your page, you need to put meta tags for title, type, image, and url in the head element.

Why is it important to separate the layout and design (css) from content (html)?

To avoid code duplication and reusability - it's easier to change the design when you don't have to change in multiple places, and it's also easier to apply the same design on new content.
